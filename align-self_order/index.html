Мы создали блок wrapper (черная граница - єто наш сайт)
мы создаем блок block (красная граница - єто наш конетет сайта)
наш конетет делится на две части (block__row (жирная желтая граница) и block__list (синяя граница))
в блоке block__row мы создаем три блока block__column (жирная виолетовая граница) + туда же добаваляем класс .block__column_1,2,3 (для кажого, чтобы могли конкретно обращатся к конкретной колонке) Пример (div)block__column .block__column_1 (/div)
в каждом блоке block__column мы создаем блок block__item (белая граница с зеленым фоном и с цифрами)

Пример 1: display: flex; (align-self)

.block__row {
display: flex;}

.block__column-1 {
align-self: center; }

.block__column-3 {
align-self: flex-end; }
align-self применяется только в flex элементу. у нее такие же функции как и в align-items.
мы дали конкретным элентами свойства. первому чтобы он был по центру, а третьему чтобы был снизу

Пример 2: комбинация (align-items и align-self)

.block__row {
display: flex;
align-items: flex-start; }
Мы всем элемента (block__column(всем трем)) внутри контейнера (block__row-контейнер) даем свойство align-items: flex-start; (прижиматся на верх)

.block__column-3{
align-self: flex-end; }
Мы же создали дополнительные классы для каждой колонки ((div)block__column .block__column_1 (/div)) и вот мы обращаемся только к третьей колонке (к третьему элементу) и даем свойство прижиматся вниз

ЗАПОМНИ: align-items - применяется только контейнерам (внутри контейнера есть элементы и они ко всем элементам применяются)
align-self - применяется только элементам video
