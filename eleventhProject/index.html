Псевдоэлемент ::before и ::after ОБЯЗАТЕЛЬНО изспользуются с свойством content: ' '; без этого свойства - псевдоэлемент не отобразиться в браузере. Свойство content ' ' - может иметь пустое значение (во внутерь кавычек ничего не добавляем), так же она может содержать нужный контент - то что мы хотим вывести (в примерах ниже)

Псевдоэлемент ::before. Применяется для отображения желаемого контента ДО содержимого элемента, к которому он добовляется



.oneExemple li::before{
content: '«'; }
