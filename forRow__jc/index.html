Мы создали блок wrapper (черная граница - єто наш сайт)
мы создаем блок block (красная граница - єто наш конетет сайта)
наш конетет делится на две части (block__row (жирная желтая граница) и block__list (синяя граница))
в блоке block__row мы создаем три блока block__column (жирная виолетовая граница)
в каждом блоке block__column мы создаем блок block__item (белая граница с зеленым фоном и с цифрами)

Пример 5: display: flex; (justify-content: space-between;)

.block__row {
display: flex;
justify-content: space-between;
}
Первый flex элемент (1) прижался к левому краю, а последний (3) к правому, дригие (в нашем случие только второй flex элемент (2)) будут размещатся между первым и последним так, чтобы между элементами были одинаковое пространство (не отступ). Если уменьшать экран, то пространство будет тоже уменьшаться так чтобы было одинаковое пространство между ними

Пример 6: display: flex; (justify-content: space-around;)

.block__row {
display: flex;
justify-content: space-around;
}
Одинаковое пространство добовляется и с лева и с права flex элемента
Мы видим что для каждого flex элемента добавились пространства: для 1 добавилось и с лево и с права. так же и остальным.
Почему не одинаковое пространство межуд 1 и 2; 2 и 3;? Потому что у первого flex элемента с права есть пространство, а у второго flex элемента есть пространство с лева - и вот эти пространство стоят рядом, значит у второго элемента два пространства и с лева и с права (видео для понимания )
