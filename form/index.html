Пример 1

Тег input принимаем данные пользователя, атрибут type это каким методом примем данные равно "text" (любой текст),
"tel" (цифрами) в мобильном устройстве автомотически переключит на цифры для ввода.
Атрубт name , каждому атрибуту type должно быть индивидуальное имя 

                                    <input type="text" name="userName" value="">                                                    <input type="tel" name="userTel" value="">

Атрибут value обычно оставляют пустым, если туда что-то вписать то получиться(смотри пример)
<input type="text" name="userName2" value="altintop">




атрибут placeholder задает подпись поля, поясняем пользователю, что это за поле
<input type="text" name="uName" value="" placeholder="введите имя">




атрибут disabled блокирует ввод. можно применят ко всем!!! заблокированые поле не передаются на сервер
< input type="text" name="uName" value="" placeholder="челси чемпион" disabled>




атрибут required - Поле обязательна для ввода

атрибут autofocus - Сразу поле выделяется и можно заполнять


< input type="text" name="user-name" required autofocus >


Атрибут type равен password для ввода паролей
<input type="password" name="enterPasswordUser" value="">

Пример 6: союзные теги.select и opition (Выпадающий список)

Тег select это коробка. Тег opition это пункты списка

<select name="color">
      <option value="red"> красный </option>
      <option value="blue"> синий </option>
      <option value="black"> черный </option>
      <option value="white"> белый </option>
</select>

атрибут seleccted по умолчанию будет выбран

    <select name="color">
          <option value="red"> красный </option>
          <option value="blue"> синий </option>
          <option value="black" selected> черный </option>
          <option value="white"> белый </option>
    </select>

атрибут multiple можно выбрать несколько цветов с зажатием ctrl

    <select name="color">
          <option value="red"> красный </option>
          <option value="blue"> синий </option>
          <option value="black"> черный </option>
          <option value="white"> белый </option>
    </select>
