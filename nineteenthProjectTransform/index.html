CSS Transform. Никак не влияет на окружение

translate (x,y).
Сдвигает элемент, перемещая относительно изначального положения
transform: translate(x, y);



box :hover {
transform: translate(10px, 20px);
} При наведении объект перемещается (грубо)
.box {
background-color: rgb(0, 255, 110);
transition: all 1s ease 0s;
}
.box :hover {
transform: translate(-10px, -20px );
} Добавили анимацию (Дальше всем)



scale (x,y).
Маштабирует элемент, делая их больше или меньше.
transform: scale (x, y);
Привет

box :hover {
transform: scale(1, 2);
} При наведении высота(у) объекта увеличивается вдвое
(1 - это ширина, по умолчанию- свой размер(150px).
2 - ширина, увеличилась вдвое (300рх) )
.box:hover {
transform: scale(1, 0.5);
}
Высота уменьшается вдвое
(0,5 - это высота, было 150рх стало 75 рх)
.scale3:hover {
transform: scale(0.5, 0.5);
} Равномерно уменьшается по высоте и ширине
.scale4:hover {
transform: scale(-1, 1);
} Отрицательное значение,
зеркальное отображение (по горизонтале)



rotate (угол+deg).
Поворачивает элементы на заданное количество градусов
(положительное число поворачивает по часовой стрелке, отрицательное против часовой)
transform: rotate (угол+deg);
