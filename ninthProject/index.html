Оригинал

Мы создаем блок (в html создаем пустой div, в css height: 200px; box-shadow: 0px 0px 10px #000; margin: 20px; border-radius: 15px; ) По умолчанию блок прозрачный, блок не имеет цвета. Чтобы не повторят код (создание блока) знай, что везде один и тот же код блока

Пример 1: Свойство background-color

background-color: blue; (блок стал синего цвета, цвет можно писать: просто blue; #0000ff; rgb; rgba-(пример ниже)) P.S. background-color: blue; = background: blue; (одинаково)

background-color: rgba(0, 0, 255, 0.3); ((0, 0, 255) - это синий цвет. 0,3 - это прозрачность. где 0 - полная прозрачность, а 1 - полная не прозрачность)

Пример 2: Свойство background-image

background-image: url(../img/squirtle.jpg); (мы подключаем одну картинку в папке img, эта картинка повторяется то тех пор пока не заполнит весь блок (выглядить так себе(ниже какая картинка нужна)), я увеличил значение height: 400px; чтобы было понятно)

background-image: url(..../img/bg_repeat.jpg); (мы подключаем другую картинку того же размера, но эта картинка вырезана так что каждый его край может быть его началом. Мы не используем большую картинку, потому что маленькая меньше весит, вес страницы будет низкой и быстрее будет работать )

Пример 3: Свойство background-repeat

background-image: url(../img/squirtle.jpg); (подключаем картинку)
background-repeat: no-repeat ; (говорим не повторять)
background: url(../img/squirtle.jpg) no-repeat; (краткая форма)

background-image: url(../img/squirtle.jpg); (подключаем картинку)
background-repeat: repeat-x ; (повторяет картинку только по горизонтали)

background-image: url(../img/squirtle.jpg); (подключаем картинку)
background-repeat: repeat-y ; (повторяет картинку только по вертикали)

Пример 4: Свойство background-position

background-image: url(../img/squirtle.jpg); (подключаем картинку)
background-repeat: no-repeat ; (говорим не повторять);
background-position: 0 50% ; (говорим его позицию, 0 - это мы двигаем по горизонтали, 50% - двигаем по вертикали (можно и в px и словами (left....)))

background-image: url(../img/squirtle.jpg); (подключаем картинку)
background-repeat: no-repeat ; (говорим не повторять);
background-position: center; (говорим его позицию, можно 50% (это значит 50% по вертикали и 50% по горизонтали))

Пример 5: Свойство background-size

background-image: url(../img/squirtle.jpg); (вставляем картинку)
background-repeat: no-repeat; (не повторяет)
background-size: 100% auto; (100% - это ширина, 0 - высота; растягиваем карти по ширине, чтобы заполнить весь блок по ширине )

background-image: url(../img/squirtle.jpg); (вставляем картинку)
background-repeat: no-repeat; (не повторяет)
background-size: auto 100%; (0- ширина, 100% - высота; растягиваем картинку по высоте, )

background-image: url(../img/squirtle.jpg); (вставляем картинку)
background-repeat: no-repeat; (не повторяет)
background-size: 700px 250px; (можем дать картинке конкретное значение (ширина-высота))

background-image: url(../img/bg.jpg); (вставляем картинку по размерам очень большая)
background-repeat: no-repeat; (не повторяет)
background-size: contain; (масштабирует картинку по меньшой стороне (по высоте))

background-image: url(../img/bg.jpg); (вставляем картинку по размерам очень большая)
background-repeat: no-repeat; (не повторяет)
background-size: cover; (масштабирует картинку по большой стороне (ширине))

background-image: url(../img/bg.jpg); (вставляем картинку по размерам очень большая)
background-repeat: no-repeat; (не повторяет)
background-size: cover; (масштабирует картинку по большой стороне (ширине))
background-position: center; (расположит картинку по центру (чтбы центр картинки был по центру))

Пример 6: Установка нескольких фоновых изображений для блока

background: url(../img/bg.jpg)0 0/ 150px 150px repeat-x, url(../img/squirtle.jpg)100% 100% no-repeat, url(../img/bg_repeat.jpg);

вставили большую картинку (расположение/поменяли размер и повторятся только по горизонтали)
вставили сквиртула (в правом нижнем углу и не повторят )
вставили третью картинку (по умолчанию повторяет пока полностью не заполнит весь блок)
