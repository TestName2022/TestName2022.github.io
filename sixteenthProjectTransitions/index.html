transition-property
(Содержит название css свойств которым будет применён эффект перехода (анимирование) )

transition-duration: 0.5s;
transition-property: all; (по умолчанию)
(все свойства анимируются)

Большая кнопка

transition-duration: 0.5s;
transition-property: background-color;
(только свойство background-color анимируется)

Большая кнопка

transition-duration: 0.5s;
transition-property: background-color, top;
(только background-color и tор анимируются)

Большая кнопка

transition-duration: 0.5s, 2s;
transition-property: background-color, top;
(мы задаем время каждому свойству отдельно )

Кнопка



transition-delay
(Это задержка выполнения перехода (анимирование) - пользволяет сделать так чтобы изменеие свойства происходило не моментально, а с задержкой (время задержки мы указываем) )

transition-duration: 0.5s;
transition-property: all;
transition-delay: 0s; (по умолчанию)

(без задержки)


Большая кнопка

transition-duration: 0.5s;
transition-property: all;
transition-delay: 2s;

(задержка на 2s после того как мы подвели курсос
и убрали курсос тоже 2s задержки)

Большая кнопка

transition-duration: 0.5s, 0.1s, 0.1s;
transition-property: background-color, top, box-shadow;
transition-delay: 0s, 1s, 3s;
(0s для background-color - нет задержки,
1s задержка для top,
3s задержка для box-shadow )

Большая кнопка









transition-timing-function
(Задаёт стиль нашего перехода, то есть некий сценарий, по которому будет осуществятся анимация.)

transition-duration: 5s;
transition-timing-function: ease;(по умолчанию)

Переход цвета начинается медленно,
разгоняется быстро и замедляется в конце

Большая кнопка

transition-duration: 5s;
transition-timing-function: linear;

Переход цвета происходит равномерно на протяжении
всего времени, без колебаний в скорости.

Большая кнопка

transition-duration: 5s;
transition-timing-function: steps(4,start);

Временная функция позволяет разбить анимацию
на чёткое количество шагов указанных
