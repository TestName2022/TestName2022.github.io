Псевдокласс focus.Срабатывает при получении элементом фокуса

.firstExemple:focus{
box-shadow: 0px 0px 10px red; }
когда щелкаешь на поле ввода, поле выделяется и понимаешь где ты заполняешь

Псевдокласс hover

.secondExemple:hover{
color: blue; }
При наведении на кнопку, слово ОТПРАВИТЬ поменяет цвет на синий + в поле для ввода текста мигающая фигня будет синего цвета (поначалу)

Псевдокласс active

.thirdExemple:active{
background-color: blueviolet; }
При нажатии на кнопку, фон кнопки поменяет цвет на фиолетовый

Комплекс

.complexInput:focus{
box-shadow: 0px 0px 10px red; }

.complexButton:hover{
